## [0.6.0] - 2020-08-25
### Added
- Added resize mode options
- Added output size options for emotes (28px, 56px, 112px) and badges (18px, 36px, 72px)
- Added an image icon to drag area
- Added status messages

### Changed
- Changed image resizing library to Jimp
